# 股票績效追蹤

即時追蹤台美股投資績效的個人理財工具。

**線上版本：** https://a5347393.github.io/stock-portfolio/

---

## 功能

- **即時報價**：台股、美股雙市場，每 60 秒自動更新
- **損益計算**：持股成本、市值、損益率即時顯示
- **資產圖表**：走勢圖（1M / 3M / 6M / 1Y / 2Y）與配置圓餅圖
- **分組管理**：自訂 4 個分組（核心、成長、配息、投機）
- **多帳戶**：支援多個投資帳戶切換與合併檢視
- **觀察名單**：追蹤有興趣但未持有的股票，可設定目標價提醒
- **交易紀錄**：買進 / 賣出歷史紀錄
- **雲端同步**：Google 登入後自動透過 Firebase 同步資料
- **CSV 匯入/匯出**：批量管理持股資料

## UX 特色

- 深色 / 淺色主題切換
- 行動裝置優化：底部導航、手機卡片視圖、Pull-to-refresh、震動回饋
- 鍵盤快捷鍵：`/` 搜尋、`N` 新增、`R` 更新報價、`?` 快捷鍵清單
- 搜尋歷史記錄（最近 5 筆）
- 目標價到達時瀏覽器通知（單次觸發，不重複打擾）
- TWD / USD 貨幣切換

## 本地端使用

直接以瀏覽器開啟 `stock-portfolio-optimized.html` 即可，無需安裝任何套件。
